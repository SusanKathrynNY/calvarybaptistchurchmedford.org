- sermons.html            Sermons
- events.html             Events
- ministries.html         Ministries overview
- resources.html          Resources
- prayer.html             Prayer Request
- blog.html               Blog
- contact.html            Contact / Plan a Visit
- give.html               Give
- sunday-school.html      Sunday School
- childrens-ministry.html Children's Ministry
- music-ministry.html     Music Ministry
- missions.html           Missions
- bible-study.html        Wednesday Bible Study
- family-ministry.html    Family Ministry
- womens-ministry.html    Women's Ministry
- mens-ministry.html      Men's Ministry
- tlc.html                Transformation Life Center

- images/                 Folder with the site's images

## Uploading to your host

Upload all files plus the `images/` folder to your hosting root, keeping the folder structure intact.

## Editing

Each page is now small (20-50KB). When you want to edit a page, just upload that one file - no need to upload the whole site.
